# Toan Tran Portfolio

Public portfolio for Toan Tran, focused on Backend/.NET engineering, enterprise
systems, release stabilization, and AI-assisted debugging workflows.

Live URL:

https://toantran0706003.github.io/

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Three.js / React Three Fiber
- Framer Motion

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run ts:check
npm run lint
npm run build
```

## Deployment

This repository deploys automatically to GitHub Pages using GitHub Actions.

- Build command: `npm run build`
- Output directory: `dist`
- GitHub Pages source: GitHub Actions

## Notes

The contact form is ready for EmailJS, but falls back to the GitHub profile when
EmailJS environment variables are not configured.

3D model licenses are preserved under `public/desktop_pc/license.txt` and
`public/planet/license.txt`.
